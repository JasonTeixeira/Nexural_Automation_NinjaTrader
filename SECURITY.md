# Security Policy

## Supported Versions
This repository is currently in early development. Security fixes will be applied to the default branch as needed.

## Reporting a Vulnerability
If you believe you’ve found a security issue:
1. **Do not** open a public issue with exploit details.
2. Contact the maintainer (Jason Teixeira / Nexural) privately via the contact method listed on the GitHub profile.

We will acknowledge receipt and work on a fix as appropriate.
